# Eventually Consistent

Custom [Ghost](https://ghost.org/) theme for [blog.rommelporras.com](https://blog.rommelporras.com), forked from [Dawn](https://github.com/TryGhost/Dawn).

See the [project README](../README.md) for development setup, deployment, and full documentation.

## Build

```bash
npm install
npx gulp build    # Compile CSS/JS → assets/built/
npx gulp zip      # Package theme → dist/eventually-consistent.zip
npx gscan .       # Validate Ghost 6.x compatibility
```
